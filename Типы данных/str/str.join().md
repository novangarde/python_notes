Метод строк, который вызывается из-под объекта типа данных строка, который будет принят в качестве разделителя.

**Синтаксис:**
```Python
<разделитель>.join(<итерируемый объект>)
```

**Пример:**
```Python
types = [type(0), type(""), type(0.0)]
output = ', '.join(t.__name__ for t in types)
print(f'[{output}]')
```

**Выведет:**


Метод join() существует в объекте string и возвращает string.
